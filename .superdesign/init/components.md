# Components

## Shared UI primitives

### `renderMonolith(career, variant)`
- File: `src/app.js`
- Purpose: 渲染职业墓碑卡片
- Visual parts:
  - status eyebrow
  - Chinese career name
  - vertical slab
  - hover quote

### `outline-button`
- File: `src/styles.css`
- Purpose: 统一 CTA 按钮风格
- Style:
  - 1px black border
  - uppercase tracking
  - black fill on hover

### `timeline`
- File: `src/styles.css`
- Purpose: 详情页和关于页的纵向时间线
- Visual parts:
  - 1px left border
  - circular markers
  - muted uppercase year label

### `factor-card`
- Files:
  - `src/app.js`
  - `src/styles.css`
- Purpose: 用于展示衰落因子、方法论卡片
